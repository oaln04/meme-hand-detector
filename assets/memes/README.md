# Meme assets

Replace these images with your own files when you want different outputs.

Expected image mapping:

- `no_hand.png` -> no hand detected
- `bite.png` -> thumb/index pinch
- `thumb.png` -> thumbs-up
- `zero.png` -> fist / 0 fingers
- `one.png` -> 1 finger
- `two.png` -> 2 fingers / peace sign
- `three.png` -> 3 fingers
- `four.png` -> 4 fingers
- `five.png` -> open hand / 5 fingers

A clearly spread thumb counts as one of the fingers. A folded thumb does not.

`.jpg`, `.jpeg`, and `.webp` also work.
